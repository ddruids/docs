### ðŸ›  Setup Instructions

#### In Figma:

1. Import the Motion Design Kit file
2. Link shared library (Tokens & Easing styles)
3. Use Smart Animate \+ Frame variants for transitions
4. Apply preset styles: `duration-short`, `ease-out-quad`, etc.

#### In Code:

```css
:root {
  --duration-short: 150ms;
  --ease-out-quad: cubic-bezier(.25,.46,.45,.94);
}
```

Use with Tailwind:

```html
<div class="transition duration-150 ease-out">...</div>
```
