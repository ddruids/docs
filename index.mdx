## Motion Tokens

Reusable variables that define animation durations and easing curves for smooth, consistent UI transitions.

#### Duration Tokens

Predefined time values that standardize animation speeds for UI motion.

| Token             | Description                                                                                    | Value |
| :---------------- | :--------------------------------------------------------------------------------------------- | :---- |
| `duration-none`   | Used for instant changes with no visual transition, ideal for opacity or binary toggles.       | 0ms   |
| `duration-xshort` | Fast, snappy feedback for microinteractions like icon hovers or small UI elements.             | 75ms  |
| `duration-short`  | Quick transitions for taps, toggles, or low-friction interactions.                             | 150ms |
| `duration-normal` | Balanced motion for tooltips, toasts, and standard UI reveals that it feels smooth but subtle. | 250ms |
| `duration-slow`   | Slower animations for larger elements or full-screen transitions add weight and clarity.       | 400ms |
| `duration-xslow`  | Deliberate pacing for major state changes or emphasis, such as modals or page shifts.          | 600ms |
